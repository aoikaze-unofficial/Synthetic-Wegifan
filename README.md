## 本地启动

1. 安装 serve 工具：

    ```bash
    npm run serve
    ```

2. 进入仓库目录，运行 `serve`：

    ```bash
    serve -p 5000
    ```
   
3. 应用程序已经运行在 http://localhost:5000
